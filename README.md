## **API**

### 一、核心

1. 画布

   + 类名：`Canvas`

   + 参数：options: Object
      - `container`：***HTMLElement*** 放置画布的容器元素
      - `scale`?：***number*** 画布缩放值
      - `maxScale`?：***number*** 最大缩放值
      - `minScale`?：***number*** 最小缩放值
      - `mode`?：***string*** 交互模式
      - `animate`?：***boolean*** 开启动画，使用连线动画时，需要设置为`true`
      - `renderType`?: ***string*** 渲染类型，默认为`DOM`用于渲染DOM类型的节点，如果要渲染canvas节点需要设置为`CANVAS`
